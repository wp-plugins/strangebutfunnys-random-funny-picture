== Description ==

Randomly displayes a funny picture from strangebutfunny.net anywhere on your site See options for help see http://anthony.strangebutfunny.net/my-plugins/sbf_random_image/ for help.

 
== Installation ==

1. Upload `the directory "strangebutfunny-random-funny-picture"` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Random Funny Pictures" in the dashboard under Settings to configure it.
